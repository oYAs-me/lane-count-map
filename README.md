# Lane Count Map
首都圏の道路を中心に、各道路の「車線数」情報を地図上に可視化・マッピングするためのデータセットを作成するプロジェクトです。

## 背景
国土交通省により、幹線道路や主要地方道に関する車線数マップは既に提供されています。
しかし、市道のバイパスや一部の主要地方道指定を受けていない高規格な道路など、公的なデータセットから漏れているものの交通上重要な路線が数多く存在します。

## 目的
既存の公的データを補完する形で、以下の達成を目指します。
- **未収録路線の整備**: 市道バイパス等、国交省データに含まれない路線の車線数情報を収集・構造化します。
- **包括的な可視化**: 既存データと統合的に扱い、道路の「太さ」やキャパシティを直感的に理解できるマップを作成します。

## 対象エリア
- 主に首都圏（東京・神奈川・埼玉・千葉）
- 必要に応じて順次拡大予定

## 想定される用途
- 交通分析の基礎データ
- ドライビングシミュレーションの道路モデル生成
- 詳細なナビゲーション補助
