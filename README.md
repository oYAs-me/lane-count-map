# Lane Count Map

首都圏の道路を中心に、各道路の「車線数」情報を地図上に可視化・マッピングするためのデータセットを作成するプロジェクトです。

## 背景

道路の「走りやすさ」やスペックを可視化する試みは過去にも存在しました。

例えば、[「走りやすさマップ」](http://www.kinokuniya.co.jp/f/dsg-01-9784904218037)のような取り組みがありましたが、現在では配布pdfのリンク切れや情報の陳腐化が進んでおり、また対象も一部の道路に限られていたりと、活用には課題があります。

一方、企業レベルでは[トヨタなどがナビゲーションシステム内で「幅広な道」「走りやすい道」を案内する機能を実装](https://toyotaconnected.co.jp/movilink/blog013.html)していますが、これらはその会社のシステム内だけでの利用に留まります。

公的なデータとしては、[国土交通省による幹線道路などの車線数マップ](https://www.mlit.go.jp/road/ir/ir-data/census_visualizationR3/webmap.html)が提供されています。しかし、これらも必ずしも全ての道路をカバーしているわけではなく、市道のバイパスや主要地方道指定を受けていない高規格な道路など、交通上重要であるにもかかわらずデータセットから漏れている路線が数多く存在します。

こうした現状を踏まえ、既存の情報を補完・統合し、地図として総覧的な車線数データを新しく整備する必要があると考えています。

## 目的

既存の公的データを補完する形で、以下の達成を目指します。

- **未収録路線の整備**: 市道バイパス等、国交省データに含まれない路線の車線数情報を収集・構造化します。
- **包括的な可視化**: 既存データと統合的に扱い、道路の「太さ」やキャパシティを直感的に理解できるマップを作成します。

## 対象エリア

- 主に首都圏（東京・神奈川・埼玉・千葉）
- 必要に応じて順次拡大予定
